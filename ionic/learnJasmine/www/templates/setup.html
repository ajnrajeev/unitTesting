 Karma CLI

 Setup for each Project These steps are required only once per project. The package.json shall contain the dependencies and an npm install is sufficient for an already setup project. Jasmine

npm install jasmine --save-dev

 Karma

Step 1: to install components, npm install karma karma-jasmine --save-dev

Step 2: To configure runner, karma init

Use default configuration for all options except files. This options specifies the files that karma needs to monitor and use. The paths for

angular.js and any other angular modules

angular-mocks.js, if needed

all application specific JS files

test spec files

are mentioned here.

Step 3: To run the tests, karma start
